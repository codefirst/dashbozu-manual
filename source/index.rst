Dashbozu User Manual
===========================================

Dashbozu is an integration service of all dashboards of the world!

Supported services are

* Jenkins
* git
* Redmine
* heroku

Requirements
-----------------------

* Play 2.0 or later
* PostgreSQL 8 or later
